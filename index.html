MLOps

This interface integrates a Flask-based application with TensorFlow Serving for real-time inference, while monitoring request volume, latency, and system health. It also incorporates a user feedback mechanism to support post-deployment evaluation and continuous model improvement.

How does it work?

  • Upload an image via the Flask interface
  • Validate file type and save the original image
  • Send the image to TensorFlow Serving via REST API
  • Perform object detection
  • Draw bounding boxes and labels on the image
  • Display results, confidence scores, and inference metrics
  • Submit user feedback to support model monitoring
